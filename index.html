(7)What are the different CSS position property values?

=>The CSS position property has five values that determine how an element is positioned in relation to the document flow

  • Static: => The default value, which positions the element according to the normal flow of the document. Placement properties like top, left, bottom, and right do not affect elements with this value.
  • Relative => The element remains in the normal flow of the document, but its position can be affected by the left, right, top, bottom, and z-index properties.
  • Absolute : => The element is removed from the normal document flow and positioned relative to its parent element
  • Fixed :=> Similar to absolute positioning, but the element is positioned relative to the viewport and will always stay in the same place even if the page is scrolled.
