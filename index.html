(1) What is CSS Box Model?

=> The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages.

(2) What is CSS selector specificity and how does it work?

=> In CSS, selectors are used to target the HTML elements on our web pages that we want to style. There are a wide variety of CSS selectors available:-

  • Inline styles: Has the highest specificity and overrides all other selectors
  • IDs: The most specific type of selector
  • Class Selector
  • Tag Selector
